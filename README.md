# Contact Form – Spring Boot & MySQL

## Description
This project is part of a Java Full Stack Internship Task. It demonstrates how a contact form collects user details from the frontend and stores them in a MySQL database using Spring Boot and Spring Data JPA.

## Features
- Contact form with Name, Email, and Message
- Spring Boot backend with REST APIs
- MySQL database integration
- Data persistence using Spring Data JPA
- Retrieve stored contact data in JSON format

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML, CSS, JavaScript

## API Endpoints
- **POST** `/submit` – Save contact form data
- **GET** `/contacts` – Fetch all stored contacts

## How to Run
1. Start MySQL server
2. Create a database named `contactdb`
3. Update database credentials in `application.properties`
4. Run the Spring Boot application
5. Open the frontend in the browser and submit the form
